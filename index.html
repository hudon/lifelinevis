Tag Propagation Graph

Here, every process thread is a node. Edges are formed between nodes that have communicated with eachother.

The different tags are shown as edges with different colors. These colors are the same as the ones used in the tree above for the same tags. Use the checkboxes below to show only certain tags.

Variations on this graph could include representing multiple connections through edge thickness as opposed to multiple edges and adding communication direction.

Thread Communication Frequency Matrix

The processes are listed on the vertical axis (sent a tag) and on the horizontal axis (received a tag). The colored squares in the graph represent the amount of communication between the source and the target process. Hovering over a square highlights the processes involved in the communication.
NOTE: Data used here differs from the above two.

Total Interations Per Process

The x-axis contains a list of all the process threads that either sent or recieved tags. The y-axis measures the total number of interactions for the tread.
